# Zaynul Aabideen Personal Website

This repository contains the source code for Zaynul Aabideen's personal website, hosted with GitHub Pages.

## Overview

The site is a lightweight static website built with:

- HTML (`index.html`)
- CSS (`main.css`)

It currently includes a basic page structure with `header`, `main`, and `footer` sections, and simple styling to scaffold the layout.

## Project Structure

- `index.html` - Main webpage markup
- `main.css` - Stylesheet for layout and colors
- `LICENSE` - Repository license
- `README.md` - Project documentation

## Run Locally

Because this is a static site, you can open it directly in your browser:

1. Clone the repository.
2. Open `index.html` in your browser.

You can also use a local dev server (recommended for live reload), for example:

```bash
python3 -m http.server 8000
```

Then visit:

`http://localhost:8000`

## Deploy with GitHub Pages

If this repository is named `zaynulaabideen.github.io`, GitHub will serve it automatically as a user site at:

`https://zaynulaabideen.github.io`

To publish updates:

1. Commit your changes.
2. Push to the `main` branch.
3. Wait for GitHub Pages to rebuild and publish.
